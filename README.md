## 🚀 Deploy on GitHub Pages

1. Create a repository named `<your-username>.github.io` (for a root URL) or any name like `portfolio`
2. Upload `index.html`, the `assets/` folder, and `.nojekyll`
3. Go to **Settings → Pages → Source: Deploy from a branch → main → / (root)** → Save
4. Your site goes live at `https://<your-username>.github.io/` (or `/<repo-name>/`)

## 🛠 Structure

- `index.html` — the entire site (HTML + CSS + JS, Three.js via CDN)
- `assets/` — project images and profile photos
- `.nojekyll` — tells GitHub Pages to serve files as-is

## ✏️ Updating

- Swap any image in `assets/` (keep the same filename) to update a project thumbnail
- All content lives in `index.html` — sections are clearly commented
